# EV Routing MILP Time-Dependent

This repository contains the MATLAB code and input datasets used in the manuscript:

"Pricing-Aware HVAC-Sensitive EV Routing and Charging Planner"

## Contents
- `code/`: MATLAB implementation
- `data/`: Excel input files used in the experiments

## Requirements
- MATLAB
- YALMIP
- intlinprog

## Main file
- `code/ev_routing_milp_time_dependent_v6.m`

## Data files
- `data/Analiz_Sonuclari_Final.xlsx`
- `data/Fixed_Weather_Data_15July.xlsx`
- `data/LMP_PJM_Data.xlsx`

## How to run
1. Open MATLAB.
2. Set the repository root as the working directory.
3. Run the main script.
